## 快速安装
1、需要安装mongodb数据库，打开数据库后数据库配置在setting.js中，在配置好数据库后可以使用node student.js来测试nodejs的连接，同时导入两个班级的测试数据。

2、确保已安装nodejs

3、在目录中 npm install

4、node app.js

5、访问127.0.0.1:3000 查看是否成功
